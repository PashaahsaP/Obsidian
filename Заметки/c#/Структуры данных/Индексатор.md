
Индексатор позволяет итерировать коллекция. Для своего типа можно также определять метод индексатор, позволяющий работать с коллекцией внутри типа.
```C#
ArrayList arPeople = new();
public Person this[int index]{
	get=>(Person)arPeople[index]
	set=>arPeople[index]=value;
}
```
Индексатор немного похож на свойство. Доступ к данным можно определять через числовое значение `int [1]` или строковое `string ["example"]`. Также можно создавать индексаторы с множеством параметров `[1,3]`. 
Также индексатор может быть определен в интерфейсе
```C#
string this[int index]{get;set;}
```

Эта ошибка возникает, поскольку типы значений копируются при назначении. При извлечении типа значения из свойства или индексатора вы получаете копию объекта, а не ссылку на сам объект. Возвращаемая копия не хранится свойством или индексатором, поскольку на самом деле они являются методами, а не местами хранения (переменными). Перед ее изменением необходимо сохранить копию в объявленной переменной.
